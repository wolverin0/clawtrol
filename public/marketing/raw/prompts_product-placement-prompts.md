# Product Placement / Image-to-Image Prompt Templates (Nano Banana / Gemini)

Objetivo: usar **screenshots reales** de nuestras apps como base y generar creativos publicitarios manteniendo la UI intacta.

## Regla de oro
Siempre separar en 3 bloques:
- **LOCK** (lo que NO puede cambiar)
- **EDIT** (lo que SÍ puede cambiar)
- **OUTPUT** (formato/estilo)

---

## 1) Template principal — LOCK + EDIT (recomendado)

**Prompt (copypaste):**

> Usá **esta imagen como BASE**.
> 
> **LOCK (no cambiar):** mantener exactamente el producto/UI principal, su forma, proporciones, texto y layout. No inventes pantallas nuevas. No deformes el contenido. No cambies logos.
> 
> **EDIT (sí cambiar):** mejorar iluminación, agregar fondo publicitario acorde a la marca, sumar sombras realistas, agregar glow sutil, añadir 2–3 callouts con flechas resaltando beneficios.
> 
> **TEXT OVERLAY:** Título: “{TÍTULO}” (blanco, grande, tipografía geométrica). Subtítulo: “{SUB}” (color {CYAN/ORANGE/BLUE}).
> 
> **BRAND:** colores {#0a1628 + #00d4ff} (o color del producto).
> 
> **OUTPUT:** 1024x1024, estilo anuncio premium, hiper legible.

---

## 2) Template — Background swap (cero riesgo)

> Usá esta imagen como base y **NO alteres el producto/UI**.
> Solo reemplazá el fondo por un background premium (degradado navy→cyan con textura tech sutil), agregá sombra suave y reflejo mínimo.
> No agregues elementos que tapen la pantalla.

---

## 3) Template — Carousel slide (para series)

> Usá esta imagen como base. Mantener la pantalla exactamente igual.
> Convertí la composición en un slide de carrusel:
> - margen superior con headline
> - margen inferior con 3 bullets (icono + texto corto)
> - 1 CTA pequeño (“Probalo gratis”)
> Estética SaaS premium, alto contraste, super legible.

---

## 4) Template — Lifestyle placement (escena realista)

> Usá esta imagen como base y mantené el UI intacto.
> Colocá el teléfono/monitor en una escena realista (escritorio moderno / cocina / gym), con iluminación natural, profundidad de campo, y sombra coherente.
> Importante: la pantalla tiene que quedar nítida y legible.

---

## 5) Template — Variantes (5 creativos de 1 screenshot)

> Usá esta imagen como base (LOCK: UI intacto).
> Generá 5 variantes cambiando SOLO: fondo, acentos de color y texto overlay.
> Variante A: {pain point} / B: {beneficio} / C: {comparación} / D: {dato} / E: {CTA}.
> Mantener composición consistente.
